
# algo-tracker

一个基于 Git 管理的算法训练仓库，用来记录我的刷题进度、错题类型、复习计划和题解代码，并配合 Claude Code 进行长期协作式刷题。

这个仓库不只是存放算法代码，还承担以下几个作用：

- 记录我刷过哪些题
- 记录每道题做错的原因
- 根据复习计划安排今天该练什么
- 沉淀常见题型模板和面试表达
- 为 Claude 提供稳定的项目上下文


每次我说开始刷题，你就要先去拉取远程的代码，然后根据我的刷题进度、错题记录、复习计划，生成一个刷题计划，并告诉我今天应该练习哪些题。

每次完成交互，根据情况决定要不要更新我的刷题进度、错题记录、复习计划。

## 仓库目标

我希望把“刷算法题”这件事从零散练习，变成一个可追踪、可复盘、可复习的长期训练系统。

这个仓库主要解决以下问题：

- 刷过的题容易忘
- 不知道哪些题该复习
- 错题没有系统记录
- 相同类型错误会反复出现
- 面试时会做题但不会表达
- Claude 每次协作都缺少历史上下文



## 快速导航

- 想看刷题进度：`data/problems.json`
- 想看今天该复习什么：`data/review_schedule.json`
- 想看自己经常错在哪：`data/mistakes.json`
- 想看本周计划：`data/weekly_plan.md`
- 想看题解代码：`problems/`
- 想看通用模板和总结：`notes/`
- 想修改 Claude 的默认协作方式：`CLAUDE.md`
